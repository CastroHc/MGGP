# Multi-Gene Genetic Programing (MGGP) toolbox


## Getting Started

### Prerequisites

To use this toolbox it is necessary the user to have installed the lybraries: numpy, scipy and deap

```
pip install deap
```

### Installing

To install the toolbox use: 

```
pip install mggp
```
